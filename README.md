# BlessingCare – Escala de Trabalho

Plataforma web para criação e gerenciamento de escalas de trabalho em regime de plantão 24 horas, com foco em simplicidade, organização e exportação de dados.

O sistema permite cadastrar colaboradores, gerar escalas automáticas em rodízio, importar uma escala anterior em Excel para manter a continuidade e exportar os resultados em Excel ou PDF.

# Funcionalidades

- Cadastro, edição e exclusão de colaboradores
- Geração automática de escala mensal em rodízio
- Continuidade da escala a partir de um arquivo Excel anterior
- Destaque automático de finais de semana
- Edição manual da escala por dia
- Exportação da escala para:
  - Excel (.xlsx)
  - PDF formatado
- Interface simples e intuitiva

# Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- [SheetJS (xlsx)] para leitura e exportação de arquivos Excel
- [jsPDF] e [AutoTable] para geração de PDF

-------

 # Como Utilizar

1. Clone ou faça o download do repositório
2. Abra o arquivo `index.html` no navegador
3. Informe o nome do cliente
4. Cadastre os colaboradores
5. Selecione o mês e o ano
6. (Opcional) Importe uma escala anterior em Excel
7. Clique em **Gerar Escala**
8. Faça ajustes manuais, se necessário
9. Exporte a escala em Excel ou PDF

# Observações

- O projeto funciona totalmente no navegador, sem necessidade de backend
- Ideal para pequenas equipes e uso administrativo
- Projeto com fins educacionais e de portfólio
